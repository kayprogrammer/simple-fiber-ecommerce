## How to run locally

* Download this repo or run: 
```bash
    $ git clone git@github.com:kayprogrammer/simple-fiber-ecommerce.git
```

#### In the root directory:
- Install all dependencies
```bash
    $ go get ./...
```
- Build and run server
```bash
    $ go run .
```
OR
```bash
    $ air
```
